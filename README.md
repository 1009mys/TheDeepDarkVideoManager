알려진 버그  
-------------  
동영상 로드하지 않고 화면 우클릭시 강제종료 현상  

ffmpeg.exe 프로세스 점유 현상  
 - 재현 : 썸네일 자동생성 -> 프로그램 강제종료  
 - 썸네일 자동생성 -> 취소 -> 프로그램 강제종료 시에는 나타나지 않음  
 
별점순 정렬 이후 정렬 초기화 현상  
태그 쓰다가 지워지는 현상  
 - 재현 : 썸네일 자동생성 -> 태그입력후 엔터를 누르지 않고 기다림  

메타데이터 설정시 다른 동영상이 영향을 받는 현상  
 - 재현 : 재현방법 필요
